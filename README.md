# Unity Background Auto-Scaler

A helpful script for Unity developers to automatically scale background sprites to fill the camera view, eliminating letterboxing issues across different aspect ratios.

## Features
- Automatically scales sprite to fit camera view.
- Supports any aspect ratio.
- Option to center the background to the camera position.
- Lightweight and easy to use.

## How to use
1. Attach the `BgCoverWorld` script to your Sprite object.
2. Ensure the object has a `SpriteRenderer` component.
3. The script will automatically scale the sprite during `Awake`.
